Scope

Scopo determina a visibilidade de alguma variavel no JS

Block statement

```JS
// vamos criar um bloco
{

    // aqui dentro é um bloco e posso colocar qualquer codigo

}   // aqui fechamos o bloco 

```

Var

    ```JS
    // var é global e podera funcionar fora de um scopo de bloco
    
    console.log('> existe x antes do bloco?', x);
    {
        var x = 0 
    }   
    console.log('> existe x depois do bloco?', x);
    ```
